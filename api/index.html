Como funciona e como faço para começar?

Nosso sistema está diariamente buscando por novos dados do CGNA. Ao receber os dados atualizados, o sistema converte os RPLs do CGNA em JSONs, formatos que são aceitos em todas as linguagens back-end e front-end (PHP, Ruby, Java, Python, NodeJS, JavaScript, etc.). Você só precisa se preocupar em gerenciar a sua VA. A parte chata, deixa com a gente.
Para começar é bastante simples, basta dar uma olhada na nossa documentação e nos exemplos.
